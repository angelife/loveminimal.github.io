日记

[2021-08-05 Thu]

5 号，周四了，一周匆匆。

新疆的项目进度太慢了…… 这个怎么收尾，也是无奈！

好长时间没有写 Vue 了，也好长时间没有关注前端技术栈的动向和趋势了。各种技术层出不穷，却万变难离其踪，上层建筑各式各样，底层结构稳定如常。要更好地理解编程语言，就需要更深入地理解计算机科学，钻研认知那些“不变”的本质。

必须是真的懂了，才是懂了！它应该是一种永恒的追求！心不会说谎！

[2021-08-04 Wed]

4 号，时间过的真快……

她又发消息了，看她过的不好，竟然还有点心疼…… 我也真是够了…… 既然不能相知相守，就相忘于江湖吧，不要再回头，不要再留恋。

Java 的核心知识，其实没有多少，面向对象是核心，封装、继承和多态。一旦心中明确知道，写出的程序是由 JVM 去弄懂的，许多先前的疑惑就自然而然地解开了。语法什么的，原来真是不是那么重要的，它只是 JVM 可以翻译的规则罢了。许多需要记忆的地方，也多是规定和约定俗成，没有什么逻辑性，多是因需而生。

基础的计算机科学是应该尽快着手用力的，内容是不少。不要怕多，慢慢地去学习、吸收、消化，利用好时间。

还是有些贪多，吃多了会撑！

想什么不是想，不要感觉不懂的太多，你都没有接触过，都没有什么认知，不懂是再自然不过的事情了。要调整好身体的节奏，不能太紧张，也不能太过放松，一张一弛，文武之道也。节奏很重要，万物运行也有其节奏，和谐于中，则必事半功倍也。

这些宝贵的知识，是不可能轻易就获得的，有些需要慢慢地去啃，一点点地去咀嚼。

要多读一些书，读好书！不然，真地成了端着金饭碗要饭了…… 自己也是要被自己笑掉大牙的了。

我们习以为常的许多事儿，都是无数仁人志士、前辈先贤的血汗换来的，务必珍惜、务必珍重！

宁可有饥饿感，也不可吃的过饱了，切记，切记！

[2021-08-03 Tue]

3 号（补、昨日聚餐，回家甚晚，未写）……

东西很多，非一朝可精通，也不符合规律啊。精通？极致？这应该是一种追求啊，为什么我感觉自己从来没有有意识地去享受过这个过程？

足够的积累，真正地去实现，不能总是想和看，虽然它们也是不可或缺的，实践是检验真理的唯一标准！具象化！把脑海中的构思付诸于实现，本身就是对其最好的再认识，思而行，查缺补漏。

原理和应用的学习不是割裂的。深入地学习原理，有利于更好、更快地理解应用的本质，是非常重要的，总是漂浮在应用的表面，就很难写出真正的好应用，稍有变动，便会觉得力不从心。然而，原理的学习注定是一个长期的、渐进的过程，不是一蹴而就的，应用做为真正的生产力工具自然不可能搁置一旁，一个剑客自然不会不能让他学会了打铁铸造再学习剑术吧。两者不是割裂的，整个过程中是不断相互砥砺的，原理的掌握是为了更好的应用，应用的升级、优化、性能的提升是对原理最好的体现和回馈。在开发应用的过程中，会对其原理有进一步的认识。两者相辅，知其然，知其所以然。

概念性的东西往往比较抽象，只有具体化了，才容易理解。把“复杂”的东西简单化，就是透过现象看本质的过程。

在进一步学习 Java 生态的过程中，颇感于其内容之多，条目之繁琐。下一步应该对核心概念进行专项突破，具体实现，仔细体会。从上学到现在，一直不怎么喜欢做练习，尤其是课后的及时练习，多数时候缘于不怎么会写，当时也不懂为什么，想当然地回避。如此想来，那时的学习方式，弊端不小。练习是对概念的再认识，是对使用场景的初步展现，是理清头脑中抽象事物的必经之路。应该加强练习，拥抱练习。

某些习惯又在蠢蠢欲动了…… 静心、体悟、为正，适当了解一下相关方面的知识，科学规律的才是正确的，不唯经验，不唯教条，实事求是。是啊，有什么东西能脱离了实事求是呢？没有！

[2021-08-02 Mon]

今天做了些什么呢？处理了一个项目问题，看了会技术网站，好像没有了…… 收获了什么呢？没有什么太大的收获，时间的利用还是不充分，Hmm，是相当的宽松了，一天就这样过去了。

今天中午吃饭的时候碰到了她，下班的时候也碰到了…… 真是奇了怪了，好久都不怎么见了，这段时间也真是见了鬼了，可能是我的时间调整的缘故。

关于她，大前天收到她发的消息了，没有回复，不知道应该回复什么，是因为太没有诚意了！有些事，匆匆忙忙的过去，对两个人都没有什么好处，毕竟都不是小孩子了，来来回回的相互纠缠了那么多年…… 到目前为止，也不好确定这件事的具体走向，也许自己内心深处还是对她有那么一丝的希望吧，只是她让人失望的次数太多了，都快麻木了。说真的，虽然有时候有些孤单，我多少有些舍不得现在这种自由的生活，可以做自己认为正确的事，可以爱家人，有大把的空余时间做自己喜欢做的事情，包括静静地一个发呆……

今天又看了一下 “Teach Yourself Computer Sicence” ，好的文章是应该多看几遍的，文章中的观点比较赞同，原理和本质性的东西再过重视也不足为过，因为那是“道”！计算机方面的书籍，还是英文版的比较丰富些，看来还是避免不了了…… 其实，自己本身也抱着读研的心思，多学些英语到是也没什么损失，只是近年来，被美国这群“自由民主的货色”搞得没了心情，必须调整一下思维了，“师夷长技以制夷”倒是个不错的出师之名。

今天没有跑步，下雨了…… 疫情近来又有反扑，它已经大大改变了人们的日常出行，每个人都在消化着这种新的生活常规。事实证明，带口罩是有益处的，感冒的人少了许多。西乡这块就算没有疫情，也没有什么适合跑步的地方，就一个西体，还有约 2km 的路程…… 很有必要找一种室内就可以进行，但又不会有太大噪音的运动方式了，以减脂、瘦腿和促进周身气血循环为核心。要不说，生活的每一个方面，如果你去多少留心一下，都有学不完的知识……

昨天的思考是对的，必须要有输出，看，只有输出，才会理清思路，才会有输入的动机。

关于哲学方面，还是要进行理论性的学习，多阅读一些政经、社会学类的书籍，想要有所作为，不了解这些是不可能的。个人、团体、众！马克思主义是迄今为止自己比较认同的哲学思维，其实别的也了解的不多，总之实事求是是没错的。

重要的不是你和别人一样或不一样，关键是你就是你，不应为了不同而标新立异，也不用为了相同就随波逐流，该不同的时候就不同，当相同的时候就相同，一切从具体实际出发就好！

明天真地要好好推进一下新疆的项目了……

[2021-08-01 Sun]

今天是八月一日喽。

到 08.20 我就整整度过三十个年头了，三十年了，就算是一个笨蛋，也会对生活有些属于自己的看法和意见了……

现在的生活状态不能说是一团糟吧，也差不了多少了。

每天都应该记录一下，从细微中扣出一丝一毫的进步，对自己来说也是一种激励吧。“逝者如斯夫，不舍昼夜。”生活应该在相同中过出不同的味道，重复但又不只是简单的重复，而应该是一种更新，以至于革新。

每天用半个小时左右的时间写一篇日记吧，记录一下思想的变化，以保持清醒。具体时间段没什么问题的话，就放在运动洗漱之后，可以听着音乐，但不要同时做其他的事情。大致也应该有几个原则，比如不能用简单的一两句话就敷衍了事，那样的话，就成了真正的流水账，没有多大意义。内容不限，但也不应该盲目，最好在下班的路上就开始对近日的行为进行自我思考和反馈。

为什么需要这样做？

因为这样就会习惯性地有所输出，从而激发输入的需求。人，一旦感觉到满了，就危险了。对事物的敏感度和新事物的好奇程度都会下降，极容易跌入停滞的深渊，而不自觉。

在经历了各种波折之后，感情的事反而对于我来说是简单的了。通过前段时间的事情，我也放下了对家人的某些负担，每个人都有自己的劫数，这也许也是属于他们的必经。对家人好，这是正确的事情，这种好应该是没有负担的、双赢的。家人可能会有一时的不开心，想不开，但这些注定只是暂时的。放下包袱，奔向新生命才是符合规律的，平时在生活上再更多地关心一些，至于自己都控制不了的事情，不应该成为一个问题。

每个人都有选择的自由，都有权利让自己感觉到快乐，这不正是应该自爱和爱人，及吾及人的理由嘛。

感谢家人给我的爱，我也会更加地爱你们，以我自己的方式。
