* 配置[fn:1] [fn:2]

新建 =~/.tmux.conf= 文件，并写入：

#+BEGIN_SRC sh -n
  # Set new default prefix
  # 修改主键
  unbind C-b
  set-option -g prefix C-j

  # Mouse
  # 激活鼠标模式
  set-option -g -q mouse on

  # Easy split pane commands
  # 修改分屏按键
  bind h split-window -h
  bind v split-window -v
  unbind '"'
  unbind %
#+END_SRC
